Quick Start

  1. Create a rifle profile and optional handload.
  2. Open Calculator, pick MIL/MOA and units.
  3. Pull weather (or enter manually). Lock heading if you want relative wind.
  4. Enter TRUE corrections as you confirm drops; TRY flows downstream.
